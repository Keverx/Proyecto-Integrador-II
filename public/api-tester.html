Eco-Flow

Simulador de Flujo Completo

Interactúa con el flujo lógico del sistema inteligente EcoScan en 4 sencillos pasos:

1
Generar PIN en Tacho IoT
Obtiene el pin temporal de un tacho registrado (Ej: Tacho ID 1).
2
Usuario Vincula Tacho
Asocia el tacho a tu cuenta ingresando el PIN que obtuviste.
3
Simular Depósito Físico
El microcontrolador del tacho envía el depósito de reciclaje (Plástico/Papel).
4
Verificar Puntos y Métricas
Consulta el Dashboard para ver tus eco-puntos incrementados en tiempo real.
